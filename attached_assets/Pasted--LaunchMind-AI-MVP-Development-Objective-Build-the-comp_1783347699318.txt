# LaunchMind AI - MVP Development

## Objective

Build the complete **New Analysis** workflow.

This is the primary workflow of LaunchMind AI.

The user journey should become:

Dashboard → New Analysis → Upload or Paste PRD → Configure Analysis → Analyze PRD

Only build the frontend.

Do not implement AI analysis yet.

---

# Rules

- Do NOT modify backend files.
- Do NOT modify Supabase.
- Do NOT connect Gemini.
- Do NOT add authentication.
- Do NOT install unnecessary packages.
- Do NOT remove existing dashboard functionality.
- Keep the current layout and navigation intact.
- Use reusable React + TypeScript components.
